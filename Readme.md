

\# 📱 Mobile Sales Dashboard (Power BI)



\## 🔍 Overview



This project presents an \*\*interactive Mobile Sales Dashboard\*\* built using \*\*Power BI\*\* to analyze sales performance, customer behavior, and product trends across different regions.



The dashboard provides a \*\*comprehensive view of mobile sales data\*\*, helping stakeholders make data-driven decisions through clear and insightful visualizations.



\---



\## 🎯 Objectives



\* Analyze \*\*overall sales and revenue performance\*\*

\* Track \*\*monthly sales and quantity trends\*\*

\* Understand \*\*customer ratings and preferences\*\*

\* Compare \*\*mobile brands and models performance\*\*

\* Evaluate \*\*payment methods distribution\*\*

\* Identify \*\*city-wise sales distribution\*\*



\---



\## 📊 Key Metrics



\* 💰 Total Sales: \*\*769.20M\*\*

\* 📦 Total Quantity Sold: \*\*19K\*\*

\* 🧾 Total Transactions: \*\*4K\*\*

\* 💲 Average Price: \*\*40.11K\*\*
